# FinZ - Simple Finance Dashboard

FinZ is a static web project developed for learning purposes. It provides a simple interface to track basic financial information, focusing on clean design and fundamental web development principles.

## 🌐 Live Demo
You can view the project here: [https://fin-z-nine.vercel.app/](https://fin-z-nine.vercel.app/)

## 🛠️ Tech Stack
This project is built from scratch without any complex frameworks, using:
* **HTML5**
* **CSS3** (Tailwind CSS)
* **JavaScript**

## 🚀 How to Run
As this is a static website, you can run it easily in two ways:

1. **Directly in Browser:**
   - Download or clone the repository to your local machine.
   - Simply open `index.html` in your preferred web browser.

2. **Using Live Server (Recommended for VS Code):**
   - Open the project folder in VS Code.
   - Install the **Live Server** extension.
   - Right-click on `index.html` and select **Open with Live Server**.

## 📝 Project Goal
This project is a simple, straightforward implementation of a finance tracker. It serves as a personal practice project to explore the fundamentals of web architecture, responsive design, and core JavaScript logic.
